# マスタメンテナンス

マスタデータ（ユーザー、コードなど）を管理する Web アプリケーション。

## 現在の状態

静的 HTML による UI プロトタイプ段階。今後 .NET Web API バックエンドを追加予定。

## 画面一覧

| 画面 | ファイル | 概要 |
|------|----------|------|
| ユーザーマスタ | `index.html` | ユーザーの一覧・検索・登録・編集・削除 |
| コードマスタ | `codes.html` | コード値の一覧・検索・登録・編集・削除 |

## 技術スタック

- HTML5 + Bootstrap 5.3.3（CDN）
- カスタム CSS（`css/style.css`）

## 開発方法

ビルド不要。HTML ファイルをブラウザで直接開く。
